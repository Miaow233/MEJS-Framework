## Medic_2.2.1 更新

1. 修复一些 Draw 的问题
2. 更新 calc 函数：

- 支持小数计算，小数部分最长保留 15 位
- 支持两个常量
  - p → π
  - e → e
- 支持更多操作符，如下：
  - % → 取余
  - ! → 乘阶
  - l → 对数 ln
  - o → 对数 log2
  - g → 开根号
  - s → 三角函数 sin
  - c → 三角函数 cos
  - t → 三角函数 tan

3. JS2 支持语音和卡片发送（详见 hdic.js）
4. JS2 新增开关机注册和执行（详见 hdic.js）
